# Flask Login System with MongoDB

This is a simple Flask application that demonstrates user authentication using MongoDB for storing user credentials. It includes login, registration, and logout functionalities.

## Installation

1. Make sure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).
2. Clone this repository to your local machine:

    ```bash
    git clone https://github.com/your-username/flask-login-mongodb.git
    ```

3. Navigate to the project directory:

    ```bash
    cd flask-login-mongodb
    ```

4. Install the required Python packages using pip:

    ```bash
    pip install -r requirements.txt
    ```

## Usage

1. Start the Flask server by running the following command:

    ```bash
    python app.py
    ```

2. Open your web browser and go to `http://localhost:5000` to access the application.

3. You can register a new account by clicking on the "Register" link on the login page. Enter your desired username, password, and email address, then click "Register".

4. After registration, you can log in with your username and password.

5. Once logged in, you will be redirected to the main page where you can view your profile or perform other actions.

6. To log out, click on the "Logout" link.

## Dependencies

- Flask
- PyMongo
